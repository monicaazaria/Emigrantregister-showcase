# 🌍 Emigrant Register

> Bachelor project developed in collaboration with Slekt og Data Telemark.

---

## 📖 About the Project

The Emigrant Register is a searchable web application that allows users to explore historical migration data from Telemark.

The solution was developed as part of my Bachelor's degree in Information Technology and Information Systems at the University of South-Eastern Norway (USN).

---

## 👩🏻‍💼 My Contribution

- Project Lead

- Full Stack Developer

- Implemented multilingual support (i18n)

- Worked with responsive design and accessibility

- Developed search functionality and user-friendly interfaces

---
